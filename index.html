Linux Presentation Foundation by AlfaOmega08

Build your impressive, double buffered, hardware accelerated, animated User Interface on Linux.

Linux Presentation Foundation is a class library used to create impressive Graphical User Interfaces. It is based on the principles that made WPF great. This is not however a plain copy of WPF although, to make the porting easier, big pieces of the .NET Framework Class Library are available with some differences. The aim of this project is to give Linux some great graphics. This project should also show the power of Linux, sometimes refused because "it's ugly".

Dependencies

Xlib Cairo Cairomm

Install

make && make install
